Setup Instructions

  1. Go to Trello Power-Ups Admin
  2. Click "Create a Power-Up" and fill in basic information
  3. Copy your API Key from the power-up settings
  4. Generate a token by visiting: https://trello.com/1/authorize?expiration=never&scope=read,write&response_type=token&name=Thunderbird%20Extension&key=YOUR_API_KEY
  5. Replace YOUR_API_KEY with your actual API key in the URL above
  6. Paste both values below and click "Save & Test"
