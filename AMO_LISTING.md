### üîß Easy Setup

1. **Get Your Trello Credentials**
   - Visit the Trello Power-Ups admin page
   - Create a new Power-Up to get your API key
   - Generate an access token with read/write permissions

2. **Configure the Extension**
   - Click the Trello icon in Thunderbird's toolbar
   - Navigate to Settings and enter your credentials
   - Test the connection to verify everything works

3. **Start Creating Tasks**
   - Select your board and list
   - Enter a task title and optional description
   - Click "Create Task" - it's that simple!

## Privacy Policy (if required)

### Data Collection
This extension does not collect, store, or transmit any personal data except:
- Trello API credentials (stored locally on your device)
- Task data sent to Trello's API (as initiated by you)

### Data Usage
- API credentials are used solely to authenticate with Trello's official API
- No analytics, tracking, or usage statistics are collected
- No data is shared with third parties

### Data Storage
- All settings and credentials are stored locally using Thunderbird's secure storage
- No remote servers or databases are used by this extension
